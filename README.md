# gr8bar
A cross-platform status bar made with Qt5

![](gr8bar/res/md/example-bar.png)
![](gr8bar/res/md/example-bar2.png)
![](gr8bar/res/md/example-bar3.png)

## Prerequisites 

### PyQt5
```
pip3 install PyQt5
```

## Trying it out
You can try out by running:

```
python3 ./gr8bar/gr8bar.py ./test/config.py
```
